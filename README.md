# 💰 Expense Tracker

A responsive Expense Tracker built using **HTML, CSS, and JavaScript**. It helps users manage their income and expenses, calculate their balance, and store transaction history using the browser's Local Storage.

## 🚀 Features

- Add Income and Expense transactions
- View Total Balance
- View Total Income
- View Total Expenses
- Transaction History
- Search Transactions
- Delete Transactions
- Print Report
- Dark Mode
- Responsive Design
- Local Storage Support (Data remains after page refresh)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## 📖 How to Use

1. Enter the transaction date.
2. Enter the description.
3. Enter the amount.
4. Select a category.
5. Choose Income or Expense.
6. Click **Add Transaction**.
7. View the updated balance and transaction history.
8. Use the search box to find transactions.
9. Delete transactions if needed.
10. Print your report using the **Print Report** button.
